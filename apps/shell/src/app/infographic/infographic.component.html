Example of webapp based on Micro-Frontend architecture.
Each one of the three apps is built and deployed independently.

Using main buttons will make remotes being discovered by Webpack module-federation plugin
and appended to Shell app's tree as children routes.
The same Material and Bootstrap remote variants can be reached as standalone
clicking on dedicated secondary buttons.

This short interactive animation should make it easier to grasp.
Hover/tap on remotes' logo to see a preview of their standalone deployment.
Then press Shell button to simulate internal routing to them.
